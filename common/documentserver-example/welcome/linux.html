Testing before integration

Use the integrated test example, a simple doc management system, to check if the editors are running correctly and examine their functionality.

Start test example:

sudo supervisorctl start ds:example

Add it to the autostart:

sudo sed 's,autostart=false,autostart=true,' -i /etc/supervisor/conf.d/ds-example.conf

Once started the example will be available here.
